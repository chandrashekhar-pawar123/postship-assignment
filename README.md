# Postship

Postship webapp.

Live Preview - [Netlify Deploy](https://postship7.netlify.app/)

## Setup Instructions

1.  Clone repository:
    ```bash
    git clone https://github.com/Pushpendra766/postship.git
    ```
2.  Install
    ```bash
    npm install
    ```
3.  Run
    ```bash
    npm run dev
    ```
